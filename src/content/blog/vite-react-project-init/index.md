Vite+React 项目初始化

## 1.使用 vite 创建项目

```bash
pnpm create vite
```

## 2.安装依赖

### 2.1 安装 sass 处理器·

```bash
pnpm i sass -D
```

### 2.2 安装 vite-plugin-chunk-split 代码拆包插件

```bash
pnpm i vite-plugin-chunk-split -D
```
